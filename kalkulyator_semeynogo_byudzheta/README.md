# Калькулятор семейного бюджета

Небольшое веб-приложение для учёта семейных доходов и расходов: счета, переводы, лимиты по категориям, цели, долги, календарь и отчёты. Интерфейс на русском языке.

## Как открыть

1. Клонируйте или скачайте репозиторий.
2. Откройте в браузере файл **`index.html`** — приветственная страница с кнопкой перехода в приложение.
3. Или сразу откройте **`app.html`** для полного функционала.

Локальный сервер не обязателен: всё работает как статические файлы (`file://` или любой статический хостинг). Файлы `app.html` и `app-i18n.js` должны лежать в одной папке.

Язык интерфейса калькулятора совпадает с выбором на главной странице (`index.html`, переключатель English / Русский). Если открыть только `app.html`, по умолчанию используется русский.

## Структура проекта

| Файл | Назначение |
|------|------------|
| `index.html` | Лендинг: краткое описание и ссылки в приложение |
| `app-i18n.js` | Словарь переводов приложения (читается из `localStorage` ключа `landingLang`) |
| `app.html` | Основное приложение (дашборд, операции, настройки и т.д.) |

Глубокие ссылки на вкладки: `app.html#dashboard`, `app.html#operations`, `app.html#settings` и т.д. (ID совпадает с атрибутом `id` у блока вкладки).

## Данные и приватность

Состояние сохраняется в **localStorage** браузера. Данные не отправляются на сервер. Рекомендуется периодически использовать **Экспорт JSON** внутри приложения.

## Разработка

Правки вносятся прямо в HTML (встроенные стили и скрипты в `app.html`). После изменений достаточно обновить страницу в браузере.

## Лицензия

Используйте свободно в личных и семейных целях.
