# 🎬 Угадай фильм по эмодзи

Интерактивная мини-игра для digital-проектов: показываем игроку набор эмодзи, он вводит название фильма, получает очки за правильные ответы и финальный результат с возможностью поделиться.

## Возможности

- 🎯 **8 раундов** из библиотеки на 12+ известных фильмов (выбираются случайно, без повторов)
- 💡 **Подсказка** для каждого раунда (−5 очков)
- ⏭ **Пропуск** раунда с раскрытием правильного ответа
- ✅ **Умная проверка ответов**: принимает русские и английские варианты, не чувствительна к регистру, пунктуации и небольшим опечаткам (Левенштейн)
- 🏆 **Финальный экран** с трофеем, вердиктом, разбором всех раундов и кнопкой «Поделиться»
- 🎉 Конфетти за правильные ответы и хороший финальный счёт
- 📱 **Полностью адаптивный** дизайн (мобильные, планшеты, десктоп)
- 🌌 Современный визуальный стиль: glassmorphism, градиенты, плавные анимации, анимированные «блобы» на фоне

## Запуск

Достаточно открыть `index.html` в любом современном браузере — никаких сборок и зависимостей.

```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

Для встраивания в digital-проект просто разместите файл на любом статичном хостинге (GitHub Pages, Netlify, Vercel, Tilda-блок «HTML», iframe и т. д.).

## Как добавить свои фильмы

Откройте `index.html` и найдите массив `MOVIES` в `<script>`. Добавьте новый объект:

```js
{
  emoji: "🎩🐰⏰",
  answers: ["алиса в стране чудес", "alice in wonderland"],
  title: "Алиса в стране чудес",
  hint: "Кроличья нора и Чеширский кот."
}
```

- `emoji` — эмодзи-ребус, который видит игрок
- `answers` — массив принимаемых вариантов ответа (русский, английский, синонимы); сравнение нечувствительно к регистру и опечаткам
- `title` — каноничное название (показывается в разборе и при пропуске)
- `hint` — текст подсказки

Если фильмов в библиотеке меньше, чем `TOTAL_ROUNDS` (по умолчанию 8), уменьшите эту константу в начале `<script>`.

## Кастомизация

Палитра задана через CSS-переменные в `:root` (`index.html`). Поменяйте `--accent`, `--accent-2`, `--accent-3`, `--bg-1..3` под бренд проекта — и весь интерфейс автоматически перекрасится.

Размер шрифта-эмодзи, скруглений, теней и блюра тоже легко настраиваются прямо в начале блока `<style>`.
